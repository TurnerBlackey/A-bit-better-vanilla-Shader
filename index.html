Installation Guide

  1. Download the shader pack ZIP file
  2. Open Minecraft and click on "Options"
  3. Go to "Video Settings" and then "Shader Packs"
  4. Click on "Open Shader Pack Folder"
  5. Move the downloaded ZIP file into this folder
  6. Select the shader pack from the list and click "Done"
  7. Enjoy!
